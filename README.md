# ⏰ Digital Watch

Welcome to **Digital Watch** – a simple, elegant and responsive project that displays the current time and date, with a built-in stopwatch and light/dark mode toggle.

## ✨ Features

- 🕒 **Live Digital Clock** – updates every second
- 📅 **Formatted Date** – weekday, day, month, and year
- ⏱️ **Stopwatch Mode** – start, pause, and reset timer
- 🌓 **Theme Toggle** – switch between dark and light mode
- 🎨 **Consistent UI** – all buttons and layout are styled uniformly
- 💻 **Responsive Design** – adapts to different screen sizes
